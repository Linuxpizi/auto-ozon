# https://api-seller.ozon.ru/v3/product/info/list

用于根据商品标识符获取商品信息的方法。

请求体应包含同类型标识符的数组，响应中将返回 items 数组。

单个请求最多可通过 offer_id、product_id 和 sku 传递总计不超过 1000个商品标识符。

## 输入
```json
{
  "offer_id": [
    "2026113PDF"
  ],
  "product_id": [
    "3339791176800"
  ],
  "sku": [
    "33373353831"
  ]
}
```
