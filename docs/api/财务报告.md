# https://api-seller.ozon.ru/v1/finance/cash-flow-statement/list

从1号到15号以及从16号到31号的财务报告获取方式。 在请求一天的报告时，您将收到15天的报告。 对应卖家个人中心财务→余额→收入和支出模块。

## 请求
```json
{
  "date": {
    "from": "2022-01-01T00:00:00.000Z",
    "to": "2022-12-31T00:00:00.000Z"
  },
  "with_details": true,
  "page": 1,
  "page_size": 1
}
```
