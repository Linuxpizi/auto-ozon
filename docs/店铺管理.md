# 店铺管理功能文档

## 店铺管理涉及的主要功能

1. 手动添加店铺
2. Excel 导入
3. 店铺列表(需要支持翻页功能)
   1. 需要展示的列
      1. 店铺名称
      2. Client ID
      3. API Key
      4. 卖家评级(小卡片)
      5. 仓库 ID
      6. 仓库状态
      7. 状态
      8. 备注
      9.  操作
          1.  编辑
          2.  同步仓库
          3.  核算
          4.  删除

## 手动添加店铺和店铺列表操作列，编辑操作
编辑操作是一个弹窗页面

弹窗标题
    店铺名称
    Client ID
    API Key
    仓库 ID
    类型 ID
    状态
    合同货币
    备注
