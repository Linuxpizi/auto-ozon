# 采集

> 临时的方案是使用 **bcs** 的官方采集插件，进行数据的采集

## 产品信息

1. SKU 的信息
2. 所有变体信息
3. 特征信息

上述的信息都是必须要采集到的，多个变体也必须要采集到，特征信息也是可以在页面直接可以采集到的

## 备选方案(当前现状)
