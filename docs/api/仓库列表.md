# [API](https://api-seller.ozon.ru/v2/warehouse/list)

该方式返回 FBS 和 rFBS 仓库列表。如要获取 FBO 仓库列表，请使用 /v1/warehouse/fbo/list 方式。

## 输入
```json
{
  "cursor": "string",
  "limit": 0,
  "warehouse_ids": [
    20605650762000
  ]
}
```
