#### 七、默认任务配置

启动时如果 `task_configs` 表为空，自动插入 3 个默认任务：

| 任务 key          | 名称     | 默认间隔 | 默认状态 | 说明                          |
| ----------------- | -------- | -------- | -------- | ----------------------------- |
| `sync_orders`     | 同步订单 | 30 分钟  | 启用     | 定时从 Ozon 拉取 FBS 订单数据 |
| `sync_finance`    | 同步流水 | 60 分钟  | 启用     | 定时从 Ozon 拉取资金报表数据  |
| `sync_warehouses` | 同步仓库 | 24 小时  | 暂停     | 定时从 Ozon 同步仓库列表      |

所有任务可在前端页面动态启用/暂停，或点击"立即执行"触发一次。配置更改通过 `PUT` API 实时同步到 APScheduler，无需重启服务。
