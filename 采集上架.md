## 二、Ozon Seller API 关键端点

### 2.1 分类树 (必须)
- `POST /v1/description-category/tree` — 获取Ozon商品分类树
  - Request: `{"language": "DEFAULT", "category_id": 0}` (0=根)
  - Response: `{"result": [{"category_id": N, "category_name": "...", "children": [...]}]}`
  - 用于：用户选择上架分类

### 2.2 分类属性 (必须)
- `POST /v1/description-category/attribute` — 获取某分类下的属性列表
  - Request: `{"description_category_id": N, "language": "DEFAULT", "type_id": N}`
  - Response: `{"result": [{"id": N, "name": "...", "description_attribute_id": N, "type": "String/Integer/Decimal/Boolean/Complex", "is_collection": bool, "is_required": bool, ...}]}`
  - 用于：展示必填属性让用户填写

### 2.3 商品上架 (必须)
- `POST /v3/product/import` — 创建/更新商品
  - Request body:
    ```json
    {
      "items": [{
        "offer_id": "你的SKU编号",
        "name": "商品标题",
        "description_category_id": 17028922,  // Ozon分类ID
        "category_id": 17036700,               // Ozon类目ID
        "type_id": 970778135,                  // Ozon类型ID
        "barcode": "6901234567890",
        "dimension_unit": "mm",
        "weight_unit": "g",
        "height": 100,
        "depth": 50,
        "width": 80,
        "weight": 300,
        "primary_image": "https://...",
        "images": ["https://...", "https://..."],
        "price": "150000",                     // 价格×100（戈比）
        "old_price": "200000",
        "vat": "0",
        "currency_code": "RUB",
        "attributes": [
          {"complex_id": 0, "id": 85, "values": [{"value": "BrandName"}]},
          {"complex_id": 0, "id": 4180, "values": [{"dictionary_value_id": 0, "value": "Китай"}]}
        ],
        "images360": [],
        "status": "processed"
      }]
    }
    ```
  - Response: `{"result": [{"task_id": N, "product_id": N}]}` (异步处理)
  - 注意：`status: "processed"` 表示自动发布

### 2.4 查上架状态
- `POST /v1/product/info/stocks` — 查库存和状态
- `POST /v3/product/info/list` — 查商品详情

### 2.5 图片上传
- Ozon要求图片为公开URL，不支持直接上传文件
- 1688图片URL有时会过期，需要先下载到自己的服务器/OSS再上传

## 三、上架流程（后端实现）

### 步骤1：选分类
1. 调 `/v1/description-category/tree` 获取分类树
2. 前端展示树形选择器（支持搜索）
3. 用户选择后，保存 `description_category_id` + `type_id` 到 scraped_product_records

### 步骤2：获取属性
1. 调 `/v1/description-category/attribute` 获取该分类下所有属性
2. 区分 `is_required` 和非必填
3. 将已有的 scraped_product_records.attributes 自动映射（按属性名模糊匹配）
4. 必填但未映射的属性标记为红色，让用户手动填写

### 步骤3：构建 import payload
```python
def build_ozon_import_item(record, category_id, type_id, attributes, offer_id):
    return {
        "offer_id": offer_id or f"AUTO-{record.source_id}",
        "name": record.title,
        "description_category_id": category_id,
        "type_id": type_id,
        "barcode": record.barcode or "",
        "dimension_unit": "mm",
        "weight_unit": "g",
        "height": record.height_mm or 100,
        "depth": record.depth_mm or 100,
        "width": record.width_mm or 100,
        "weight": record.weight_g or 500,
        "primary_image": record.images[0] if record.images else "",
        "images": record.images[:10] if record.images else [],
        "price": str(int(record.price * 100)) if record.price else "0",
        "old_price": str(int(record.old_price * 100)) if record.old_price else "",
        "vat": "0",
        "currency_code": "RUB",
        "attributes": attributes,
        "status": "processed",
    }
```

### 步骤4：调用上架
1. 调 `POST /v3/product/import`
2. 拿到 `task_id`，存入 listing 表
3. 轮询 `/v1/product/info/stocks` 查状态
4. 上架成功后更新 scraped_product_records 的 `matched = True`

## 四、价格换算规则

```
1688 人民币 → Ozon 卢布：
  成本价(¥) × 汇率(约12.5) × 加价系数(1.3~2.0) = Ozon售价(₽)
  还需考虑：Ozon佣金(5~15%) + 物流费 + 包装费

简化方案：
  Ozon售价 = 1688价格 × 汇率 × 倍率
  默认汇率: 12.5, 默认倍率: 1.5
  用户可在前端调整
```

- `GET /selection/ozon-attributes` — 获取分类属性

scraped_product_records 表新增：

## 六、前端实现计划

### 6.1 上架对话框 (UploadDialog.vue)
1. **Step 1 - 选择店铺**：下拉选择已配置的 Ozon 店铺
2. **Step 2 - 选择分类**：树形选择器 + 搜索（调 /selection/ozon-categories）
3. **Step 3 - 填写属性**：必填属性高亮，自动填充已有数据（调 /selection/ozon-attributes）
4. **Step 4 - 确认上架**：显示最终数据预览，确认后调 /upload

### 6.2 价格计算器
- 输入 1688 价格(¥) → 自动计算 Ozon 售价(₽)
- 可调参数：汇率、倍率、目标利润率

### 6.3 上架状态追踪
- 上传后在列表中显示状态标签（pending/success/failed）
- 失败时显示错误原因
