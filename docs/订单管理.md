# 订单管理功能文档

对授权的所有店铺订单信息进行同步管理

## 订单列表

1. 检索条件
   1. 店铺名称
   2. 货件状态
   3. 货件号/订单号/运单号
   4. 刷新
   5. 一键同步
   6. 总数(只展示)
2. 订单列表(支持翻页)
   1. 图片/货件号/SKU
   2. 订单号
   3. 状态/操作
   4. 必须发货/商品名称
   5. 运单号/数量/成交价
   6. 店铺/账号/极速下单
