# https://api-seller.ozon.ru/v1/finance/balance

对应卖家个人中心 财务 → 余额 模块。

您可以在 讨论 的评论中对此方法提供反馈 在 Ozon for dev 开发者社区中。

报告期结束日期，格式为 YYYY-MM-DD。date_from 与 date_to 之间的最⻓间隔为30 天。

## 请求
```json
{
  "date_from": "2019-08-24",
  "date_to": "2019-09-24"
}
```
