## 一、定价策略 (PricingStrategyAPI)

### 核心概念
- **定价策略 (Strategy)**: 绑定一组竞争对手和系数，自动跟踪竞品价格
- **竞争对手 (Competitor)**: Ozon 上的其他卖家，每个有唯一 `competitor_id`
- **系数 (Coefficient)**: 跟踪竞品价格的倍率（1 = 同价，0.95 = 低于竞品 5%）

### API 端点

| 方法 | 端点 | 说明 |
|------|------|------|
| GET | `/v1/pricing-strategy/list` | 获取定价策略列表 |
| GET | `/v1/pricing-strategy/info` | 获取策略详情（含竞争对手列表） |
| POST | `/v1/pricing-strategy/create` | 创建新策略 |
| POST | `/v1/pricing-strategy/update` | 更新策略名称和竞争对手 |
| POST | `/v1/pricing-strategy/delete` | 删除策略 |
| POST | `/v1/pricing-strategy/status` | 启用/禁用策略 |
| GET | `/v1/pricing-strategy/competitors/list` | 获取可选竞争对手列表 |
| GET | `/v1/pricing-strategy/products/list` | 获取策略关联商品 |
| POST | `/v1/pricing-strategy/products/add` | 添加商品到策略 |
| POST | `/v1/pricing-strategy/products/delete` | 从策略移除商品 |
| POST | `/v1/pricing-strategy/product/info` | 获取单个商品的竞品价格 |
| POST | `/v1/pricing-strategy/strategy-ids-by-product-ids` | 查询商品绑定的策略 |

### 数据结构

**策略详情响应**:
```json
{
  "result": {
    "strategy_name": "我的策略",
    "enabled": true,
    "update_type": "strategyItemsListChanged",
    "type": "COMP_PRICE",
    "competitors": [
      { "competitor_id": 204, "coefficient": 1 },
      { "competitor_id": 1008426, "coefficient": 1 }
    ]
  }
}
```

**竞品价格响应**:
```json
{
  "result": {
    "strategy_id": "uuid",
    "is_enabled": true,
    "strategy_product_price": 114,
    "price_downloaded_at": "2022-11-17T15:33:53.936Z",
    "strategy_competitor_product_url": "https://..."
  }
}
```

## 二、Ozon 平台促销活动 (Promos / PromosBeta)

### 核心概念
- **活动 (Action)**: Ozon 平台发起的促销（如 Hot Sale、节日促销）
- **参与状态**: 卖家可选择参加或不参加
- **活动价格 (Action Price)**: 参加活动时商品的促销价格

### API 端点

| 方法 | 端点 | 说明 |
|------|------|------|
| GET | `/v1/actions` | 获取可参加的平台促销活动列表 |
| GET | `/v1/actions/products` | 获取活动中的商品列表（分页） |
| POST | `/v1/actions/products/activate` | 添加商品到活动（设定活动价格） |
| POST | `/v1/actions/products/deactivate` | 从活动中移除商品 |

## 三、卖家自定义促销活动 (SellerActions)

### 核心概念
- **卖家活动**: 卖家自己创建的折扣活动（限时折扣、优惠券等）
- **活动类型**: DISCOUNT（折扣）、GIFT（赠品）、VOUCHER（优惠券）
- **预算控制**: 可设置活动预算和已花费金额

### API 端点

| 方法 | 端点 | 说明 |
|------|------|------|
| GET | `/v1/seller/actions` | 获取卖家促销列表 |
| POST | `/v1/seller/actions` | 创建卖家促销 |
| PUT | `/v1/seller/actions/{action_id}` | 更新促销参数 |
| DELETE | `/v1/seller/actions/{action_id}` | 删除促销 |
| GET | `/v1/seller/actions/{action_id}/products` | 获取促销中的商品 |
| POST | `/v1/seller/actions/{action_id}/products` | 添加商品到促销 |
| DELETE | `/v1/seller/actions/{action_id}/products` | 从促销移除商品 |

## UI 设计方案

### 页面结构
```
/intelligence (智囊总览)
├── PricingStrategy   (定价策略管理)
├── PlatformPromos    (平台促销活动)
└── SellerPromos      (卖家自定义促销)
```

### 1. 智囊总览页 (/intelligence)
- **顶部卡片**: 3 个核心指标
  - 活跃定价策略数
  - 可参加的平台活动数
  - 活跃卖家促销数
- **快捷入口**: 跳转到三个子模块

### 2. 定价策略管理页 (/intelligence/pricing)
- **策略列表**: 表格展示所有策略（名称、状态、竞争对手数、商品数）
- **操作**: 创建、编辑、启用/禁用、删除
- **详情面板**: 点击策略查看关联商品和竞品价格
- **竞品分析**: 展示每个商品的竞品价格对比

### 3. 平台促销活动页 (/intelligence/platform-promos)
- **活动列表**: 可参加的 Ozon 促销活动
- **筛选**: 进行中 / 即将开始 / 已结束
- **操作**: 参加活动、设置活动价格
- **商品管理**: 查看和管理活动中的商品

### 4. 卖家促销活动页 (/intelligence/seller-promos)
- **促销列表**: 卖家创建的自定义促销
- **创建向导**: 引导创建新促销（类型、时间、折扣、预算）
- **商品管理**: 添加/移除促销商品
- **预算监控**: 显示预算使用情况
