## 实现方案

### 1. SKU 复制精铺

#### 流程
1. 输入源 SKU(Ozon 平台商品)
2. 调用 API 获取商品完整信息
3. 翻译处理:
   - 商品名称(俄语 → 目标语言)
   - 商品描述
   - 变体特征
4. 图片处理:
   - 下载原图
   - 可选:添加水印/翻译文字
5. 编辑优化后的信息
6. 创建新商品(草稿状态)
7. 提交审核

#### API 调用链
```
1. POST /v3/product/info/stocks    # 获取库存信息
2. POST /v4/product/info/list      # 获取商品列表详情
3. POST /v1/product/pictures/upload # 上传图片
4. POST /v2/product/import         # 创建商品
```

### 2. 链接爬取精铺

#### 流程
1. 输入商品链接(淘宝/1688/拼多多等)
2. 爬取商品信息:
   - 基本信息(标题、价格、规格)
   - 图片列表
   - SKU 变体
3. AI 翻译处理
4. 图片下载与处理
5. 手动编辑优化
6. 创建 Ozon 商品

### 数据模型

```python
# 精铺任务
class PrecisionListingTask:
    id: int
    store_id: int                    # 目标店铺
    source_type: str                 # "sku_copy" | "link_scrape"
    source_value: str                # SKU 或链接
    status: str                      # "pending" | "processing" | "editing" | "completed"
    
    # 原始数据
    original_data: dict              # 原商品信息
    
    # 处理后数据
    translated_name: str
    translated_description: str
    processed_images: list           # 处理后的图片URL
    
    # 最终数据
    final_data: dict                 # 提交前的完整数据
    ozon_product_id: int             # 创建后的商品ID
    
    created_at: datetime
    updated_at: datetime
```

## 图片处理策略

| 处理类型 | 说明 | 优先级 |
|---------|------|--------|
| 尺寸裁剪 | 调整为 Ozon 推荐尺寸 | 必需 |
| 背景去除 | AI 自动抠图 | 可选 |
| 水印添加 | 添加店铺 Logo | 可选 |
| 文字翻译 | 图片内文字翻译 | 按需 |

## 注意事项

1. **图片要求**
   - 最少 5 张图片
   - 主图白底
   - 分辨率不低于 200x200
   - 文件大小不超过 5MB

2. **文案要求**
   - 标题长度 15-150 字符
   - 避免敏感词
   - 包含核心关键词

3. **价格策略**
   - 考虑平台佣金
   - 预留促销空间
   - 参考竞品定价
